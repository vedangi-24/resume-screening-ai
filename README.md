# AI Resume Screening System

AI-powered ATS Resume Screening System built using Flask.

## Features

- Resume PDF Upload
- ATS Score Calculation
- Skill Matching
- Missing Skills Detection
- Candidate Ranking
- AI Resume Suggestions
- ATS Report Download

## Tech Stack

- Python
- Flask
- PyPDF2
- HTML
- CSS

## Run Locally

pip install -r requirements.txt

python app.py
